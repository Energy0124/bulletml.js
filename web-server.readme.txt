=simple-web-server.jar=

==概要==

HTTPリクエストを待ち受けてリソースを返すだけのシンプルなWebサーバです。
Webアプリ開発に役立ててください。

起動すると8080番ポートを待ち受けます（デフォルト時）。
起動フォルダにある「htdocs」をドキュメントルートとして動作します（デフォルト時）。

==動作環境==

Java1.6以上。

==使い方==

1.起動方法
    以下のコマンドを実行

        java -jar simple-web-server.jar [-p <port>] [-d <docRoot>]

2.終了方法
    「Webサーバ停止」ボタンをクリックする。

3.起動オプション
    -p <ポート番号> 待ち受けポート番号を指定します。デフォルト値は8080です。
    -d <ドキュメントルート> ドキュメントルートを指定します。デフォルト値は"htdocs"です。
